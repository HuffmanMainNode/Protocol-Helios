# llnl/GRACE---GRinding-Automated-Classification-Engine

**Description:** Machine learning classifier for quality control of alloy sample grinding. Uses a YOLO based classifier on microscopy images to perform binary classification of grind quality, detecting surface scratches and separating pass or fail samples for an autonomous preparation workflows.
