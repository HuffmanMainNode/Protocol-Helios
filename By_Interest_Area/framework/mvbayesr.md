# sandialabs/mvBayesR

**Description:** R implementation of the multivariate Bayesian regression (mvBayes) framework. Decomposes a multivariate/functional response using a user-specified orthogonal basis decomposition, and then models each basis component independently using an arbitrary user-specified Bayesian regression model. Includes prediction and plotting methods.
